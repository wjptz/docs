Suncode is a repo-native workflow layer for AI coding assistants. It automatically injects your project's specs, coding conventions, and workflow rules into every AI session — so agents write code that follows your standards instead of improvising from scratch. Works across Claude Code, Cursor, Codex, Kiro, Gemini CLI, and 12+ other platforms.

## Why Suncode?

AI coding tools are powerful, but they forget your project between sessions and improvise conventions from scratch every time. Suncode solves this by keeping engineering rules in the repo — versioned, shared, and auto-injected.

## Get Started in 3 Steps

<Steps>
  <Step title="Install Suncode">
    Install the global CLI via npm:
    ```bash
    npm install -g @mindfoldhq/trellis@latest
    ```
  </Step>
  <Step title="Initialize your project">
    Navigate to your project directory and run init with your developer name and chosen platform:
    ```bash
    cd your-project
    trellis init -u your-name --claude
    ```
  </Step>
  <Step title="Start your first task">
    Open your AI assistant and describe what you want to build. Suncode automatically injects context, tracks your task, and guides the AI through Plan → Execute → Finish.
  </Step>
</Steps>

<Tip>
  Not sure which platform flag to use? Run `trellis init -u your-name` without any platform flags and Suncode will auto-detect the AI tools installed on your machine.
</Tip>
